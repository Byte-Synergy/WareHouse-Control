# 🚀 Quick Start Guide

## StockMaster Lite - 5分でスタート

### ステップ 1: インストール
```bash
npm install
```

### ステップ 2: サーバー起動
```bash
npm start
```

### ステップ 3: ブラウザでアクセス
```
http://localhost:3000
```

### ステップ 4: ログイン
- **ユーザー名:** admin
- **パスワード:** admin123

---

## 🎯 最初にやること

### 1. 商品を追加する
1. 「+ 商品追加」ボタンをクリック
2. SKU（例: PROD-001）と商品名を入力
3. 保存をクリック

### 2. 在庫を管理する
1. 商品の「在庫操作」ボタンをクリック
2. 「入庫」または「出庫」を選択
3. 数量を入力して実行

### 3. 低在庫を確認する
- 画面上部のアラートをチェック
- 「低在庫のみ表示」ボタンでフィルター

### 4. データをエクスポート
- 「CSV出力」ボタンをクリック
- Excelで開ける形式でダウンロード

## ⚠️ 重要事項

### 本番環境での使用前に:
1. ✅ デフォルトパスワードを変更
2. ✅ HTTPS を設定
3. ✅ 定期的なバックアップを設定

### データバックアップ:
```bash
# データベースファイルをコピー
cp inventory.db inventory-backup.db
```

---

## 🛠️ トラブルシューティング

### サーバーが起動しない
```bash
# ポート3000が使用中の場合
PORT=3001 npm start
```

### データベースをリセット
```bash
rm inventory.db
npm start
# 自動的に新しいデータベースが作成されます
```

---

## 💡 ヒント

- **SKU:** 一貫性のある命名規則を使用（例: CAT-001）
- **最小在庫:** 商品の回転率に応じて設定
- **バックアップ:** 週次でCSVエクスポートを実行
- **低在庫:** 定期的にアラートをチェック
